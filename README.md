# Agency portfolio
 
Practice portfolio in React

![Agency portfolio](design/preview.png "desktop-preview")

## Installation

```
git clone git@github.com:donCelis/agency-react.git
cd agency-react
npm i
```

## Usage

### -Development server

```bash
npm run start
```

You can view the development server at `localhost:3000`.

### -Production build

```bash
npm run build
```

### -Deploy Github Pages

```bash
npm run deploy
```
